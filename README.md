## 🚀 About

**zar** is the definitive multi-purpose Discord application designed for elite servers. It combines high-end moderation, advanced utilities, and powerful engagement tools into a single, seamless experience.

Trusted by **300+ servers** and **140,000+ users**, zar provides the infrastructure you need to build and scale a world-class community.

## 📦 Modules

zar comes packed with everything you need, built right in:

### 🛡️ Moderation & Security
- **Auditlogs:** Track every action within your server with precision.
- **Moderation:** Powerful tools to keep your community safe.
- **Logs:** Keep a detailed record of server events.

### ⚙️ Management & Utility
- **Autorole:** Automatically assign roles to new members.
- **Role Management:** Manage roles and permissions easily.
- **Utility:** Essential utilities for server management.
- **Info:** Display detailed server and user information.
- **Emoji:** Manage and steal emojis for your server.

### 🎉 Engagement & Fun
- **Giveaway:** Host and manage giveaways seamlessly.
- **Games:** Fun mini-games to keep chat active.
- **Counting:** Engage members with counting channels.
- **Greetings:** Customizable welcome and goodbye messages.

### 🔧 Advanced Tools
- **Custom Embeds:** Create beautiful custom embeds easily.
- **PVC (Private Voice):** Allow members to create temporary private voice channels.
- **Reminder:** Set reminders for important tasks.
- **Premium:** Unlock exclusive features and limits.
